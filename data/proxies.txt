# Proxy list for email crawler
# One proxy per line. Supported formats:
#   http://host:port
#   http://user:pass@host:port
#   socks5://host:port
#   socks5://user:pass@host:port
#
# Lines starting with # are ignored.
# Dead proxies are automatically removed after 3 failures.
# Each proxy is cooled down for 60s after a failure.
#
# Examples:
# socks5://127.0.0.1:1080
# http://user:pass@us-proxy.example.com:8080
# http://user:pass@in-proxy.example.com:8080
